Projets Ufomatics

  • Archipelago - Une meta-base pour aggréger les bases de données ufologiques existantes
  • UFO@home - Un outil pour témoigner visuellement d'observations d'ovnis
  • Applet de statistiques - Un petit programme calculant et affichant des statistiques d'observations.
  • Matrice de phénomènes connus - Un petit programme calculant les explications OVI les plus probables sur la base de réponse à un ensemble de questions
  • Stellarium for Java - Un version Java de Stellarium, destinée à constituer la base de la prochaine génération plus réaliste de UFO@home.

Autres logiciels d'ufologie

  • UFO tools de Terry Groff - Formules en Javascript aidant à diverses évaluations et calculs nécessaires dans les enquêtes ufologiques
  • *U*Database dey Larry Hatch - Une des bases de données ufologiques les plus connues
  • SonotaCo
    • UFOCapture - Logiciel de capture de mouvement décalée dans le temps
    • UFOAnalyzer - Outils de post-traitement pour UFOCapture
    • UFOOrbit - Détecteur d'observations simultanées / calculateur d'éléments orbitaux
  • ReactivUfo - Pilotage d'une station d'enregistrement automatique.

Logiciels d'intérêt pour l'ufologie

  • Stellarium - Une planetarium réaliste depuis n'importe quel endroit
  • Sunrise, Sunset calendars and Local Time - Ephemerides calculées pour le Soleil et la Lune, depuis n'importe quel endroit à n'importe quel date
  • Worldwind par la NASA - Carte 3D interactive de la Terre
    • EarthFlicks, un module Java permettant d'afficher des animations SVS sur un globe 3D
  • J-Track par la NASA - Suivi de satellites
