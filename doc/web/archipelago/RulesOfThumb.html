Evaluation is ufologists' job

As a tool, Archipelago does not intend to assess data by itself. This means that it does not contain hardcoded rules to select data or data sources based on some arbitrary criteria.

However, it allows the user to freely select the data of interest to him, based on his/her own customized criteria.

Sources are critical

Data without sources is nothing but rumour. As Archipelago find its data in another databases, such an origin is, more than ever, a critical information that will always be available to the user. Every data (as deeply as data field level) retrieved through Archipelgo has a source information.
