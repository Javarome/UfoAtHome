Generally speaking, a database is a set of structured data. Such a structure allow to apply processes on it, the most common of which being the search of some data given some structured criteria.

Concretly, software databases can adopt a lot of different structures and use a lot of different technologies. These can range from flat files to relational database (like MySQL), including Excel files or structured web pages.

More and more databases are becoming available "on line", that is, reachable through the Internet. The Archipelago project aims to allow the UFO researcher to access all those databases through a common virtual view that meets its needs (suited to the processes he wrote typically).
