L'évaluation est le travail des ufologues

En tant qu'outil, Archipelago n'entend pas évaluer les données en elles-mêmes. Cela signifie qu'il ne contient pas des règles figées de sélection de données ou de sources de données sur la base de tels ou tels critères arbitraires.

Cependant, il permet à l'utilisateur de sélectionner librement les données qui l'intéressent, sur la base de ses propres critères personnalisés.

Les sources sont critiques

Une donnée sans source n'est rien d'autre qu'une rumeur. Archipelago trouvant ses données dans d'autres bases de données, une telle origine est, plus que jamais, une information critique qui sera toujours accessible à l'utilsateur. Toute donnée récupérée via Archipelago (aussi bas que le niveau d'un champ) a une information de source.
