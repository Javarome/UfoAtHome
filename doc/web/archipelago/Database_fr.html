Bases de données

Au sens général, une base de données est un ensemble de données structurées. Une telle structure permet d'appliquer des traitements sur ces dernières, le plus courant d'entre eux étant la recherche de données sur la base de critères structurés.

Concrètement, les logiciels de base de données peuvent adopter nombre de structures différentes et utiliser nombre de technologies différentes. Celles-ci peuvent varier de fichiers plats à aux bases de données relationnelles (comme MySQL), en passant par des fichiers Excel ou des pages web structurées.

De plus en plus de bases de données deviennent disponibles "en ligne", c'est-à-dire, atteignable via l'Internet. Le projet Archipelago vise à permettre au chercheur sur les ovnis d'accéder à l'ensemble ces bases de données à travers une vue virtuelle commune qui réponde à ses besoins (adaptées aux processus qu'il a écrit typiquement).
