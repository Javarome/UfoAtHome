Au sens strict, des meta-données sont des données sur des données. C'est-à-dire, une manière de décrire des données (leurs sémantique, relations avec d'autres données, et ainsi de suite).

Archipelago utilise 3 types de metadonnées :

  1. le schéma de base de données, qui décrit la structure d'une base de données existante et la manière dont on peut y accéder.
  2. le meta-modèle, qui décrit la structure de données que veut l'utilisateur veut manipuler. C'est-à-dire, la structure de la vue à travers laquelle les données concrètes doivent être visualisées.
  3. le mapping de base de données, qui décrit la correspondance entre les 2.

L'implémentation de référence du projet Archipelago fournira :

  • des schémas de bases de données pour diverses bases en ligne connues
  • un exemple méta-modèle
  • des mappings de ces bases de donneés vers ce meta-modèle
