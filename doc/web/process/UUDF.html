Uniform Ufological Data Format : format uniforme de données ufologiques.

Besoin

Définir un ensemble de données permettant de :

  • stocker tout cas ufologique
  • classer tout cas ufologique dans les principaux systèmes de classification connus

Analyse

UUDF définit les propriétés suivantes pour un cas :

  • lieu
    • type (plan d'eau, sol, urbain)
    • latitude et longitude
    • bourg, ville (région, pays)
    • traces au sol
  • temps
    • date
    • heure
    • temps manquant
  • objets observés
    • élévation (azimut)
    • nombre
    • forme(s)
    • trajectoire(s) et comportement(s)
    • luminosité
    • couleur(s)
    • texture(s)
  • témoin(s)
    • orientation
    • nom
    • prénom
    • adresse
    • téléphone(s)
    • email
    • séquelles
      • physiques
      • psychologiques
  • conditions météorologiques
