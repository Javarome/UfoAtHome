Setup

Prérequis

UFO@home nécessite :

  • un conteneur Web conforme à J2EE 1.2, impliquant le support des API :
    • Servlet 2.2
    • JSP 1.1
  • une base de données compatible SQL 92 avec un pilote conforme à JDBC 2.0.

Les configurations testées sont :

Plate-forme Serveur applicatif Serveur Web Base de données
Native Java Serveur Pilote JDBC
           

L'installation nécessite l'outil Ant. Si vous n'avez pas Ant installé, vous pouvez le télécharger depuis le site Web de Jakarta.
