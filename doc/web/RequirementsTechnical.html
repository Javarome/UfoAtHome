Portabilité

L'application doit pouvoir être déployée sans modification de code sur le plus grand nombre de plates-formes différentes.

Les technologies prévues sont :

  • une plate-forme J2EE avec support du déploiement sur :
    • Apache Tomcat
    • JavaSoft J2EE RI 1.3
    • IBM WAS 4.03
  • Couche de persistance pluggable
    • JDBC : SQL92-compatible RDBMS, avec support de :
      • MySQL 3.23
      • Oracle 8.1.7
    • JDO, avec support de :
      • Apache OBJ
      • Libelis LiDO
