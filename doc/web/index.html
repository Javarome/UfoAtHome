Ufomatics projects

  • Archipelago - A meta-framework to aggregate existing ufological databases
  • UFO@home - A tool for reporting UFO sightings visually
  • Statistics applet - A small program that computes and display statistics about sightings
  • Known phenomenon matrix - A small program that computes most probable IFO explanations given answers to a set of questions
  • Stellarium for Java - A Java version of Stellarium, aimed to be the basis of the next, more-realistic generation of UFO@home

Other ufology software

  • UFO tools by Terry Groff - Javascript formulas that help to various evaluations and computations required in UFO investigations
  • *U*Database by Larry Hatch - One of the best known database of UFO cases
  • SonotaCo
    • UFOCapture - Time Shifted Motion Capture Software
    • UFOAnalyzer - Post processing tool for UFOCapture
    • UFOOrbit - Simultaneous observation detector / orbital elements calculator
  • ReactivUfo - Driving an automatic recording station.

Software of interest for ufology

  • Stellarium - A realistic planetarium from any location
  • Sunrise, Sunset calendars and Local Time - Computed ephemerids of Sun and Moon, from any location at any time
  • Worldwind by the NASA - A 3D interactive map of the Earth
    • EarthFlicks, a Java application that allows displaying SVS animations on a 3D globe
  • J-Track by the NASA - Satellites tracking
  • ISpy Connect - A Camera Security Software initially designed for ghost hunting and UFO spotting. You can trigger recording on the slightest sound or movement. iSpy also captures preview frames at the moment of maximum movement to enable you to quickly sort through a lot of files to find anything interesting.
