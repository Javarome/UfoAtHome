Bienvenue sur la page d'accueil de UFO@home.

UFO@home est un projet de logiciel Open Source visant à fournir un système d'information complet pour standardiser/unifier l'échange, le traitement et le format des données ufologiques, ainsi que la manière d'y accéder.
