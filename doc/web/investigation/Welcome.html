RR0 Information System (Système d'Information RR0) est un projet de logiciel OSS visant à fournir un système d'information complet pour :

  • standardiser/unifier le format des données ufologiques
  • centraliser les données ufologiques mondiales pour permettre aux chercheurs de partager et consolider les données
  • traiter ces données de manière automatisée afin d'obtenir statistiques, contrôle de cohérence, recherche de similarités de comportement, de temps, de lieu, etc.

En raison du public visé à travers le monde et de son but d'unification, le projet a été placé dans la communauté Open Source en Décembre 2002. Le contenu du projet est en permanence disponible sur SourceForge.net sous licence Apache 1.1, permettant à tous de visualiser, utiliser, modifier et redistribuer son code, moyennant qu'il soit toujours livré avec sa licence.
