Ajout d'une classification

Une classification définit un ensemble de codes dont chacun est associé à un ensemble de valeurs pour un ensemble de critère d'observation. Elle permet de regrouper les observations en ensembles caractéristiques.

Afin d'être de ne pas créer une classification déjà existante ou très proche d'une classification existante, n'hésitez pas à consulter les classification actuellement définies :

Classification

Dans le cas où votre classification n'est pas semblable à une classification définie ci-dessus, vous pouvez commencer à définir votre nouvelle classification.
