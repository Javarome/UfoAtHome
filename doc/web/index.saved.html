Welcome to the UFO@home project home page.

UFO@home is an Open Source Software project that aims to provide a full information platform for standardizing/unifying exchange, processing and formats of ufological data, as well as the way to access them.

UFO@home is composed of 3 subprojects :

  • Sighting : provide a standardized and efficient way to report and replay UFO sightings.
  • Process : provide a way to apply processing (statistics, checking) on data manageable by UFO@home servers
  • Investigation : provide tools for organization and real-time follow-up of UFO investigations all over the world.
